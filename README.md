# defect-detection-cnn-vs-yolo
Проект посвящен разработке программно-аппаратного комплекса для интеллектуального контроля качества ПЭТ-преформ непосредственно на конвейере. Вместо стандартного YOLO-подхода для задачи детекции дефектов была предложена и реализована CNN-модель классификации, показавшая стабильные результаты при постепенном расширении датасета и числа классов.

Нейро-контроль преформ: система автоматизированного визуального мониторинга. Применение CNN вместо YOLO. 

О проекте

Проект выполнен в рамках задания:
https://ai-hunter.ru/tbr2025

В исходной постановке предполагалось использование YOLO для задачи детекции дефектов на преформах.

Однако в рамках проекта было предложено альтернативное решение:
использовать CNN-модель классификации вместо detection-подхода.

Почему CNN вместо YOLO

- задача может быть сведена к классификации;
- не требуется разметка bounding boxes;
- проще обучение и интеграция;
- устойчивость к неполным данным.

Основные сложности:

- ограничения Google Drive на загрузку файлов;
- частичная потеря данных при скачивании;
- несбалансированные классы при скачивании предыдущих файлов с меньшим количеством изображений;
- постепенное наращивание датасета.

Принятые решения:

- переиспользование ранее загруженных данных;
- поэтапное обучение моделей;
- аугментация малочисленных классов;
- финальная сборка полного датасета.
